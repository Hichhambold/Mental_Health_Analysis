# Mental Health Analysis

Ziel ist, die Auswirkung des digitalen Verhaltens auf die mentale Gesundheit zu untersuchen.


## Start
```bash
pip install -r requirements.txt
```

Mittels des Tools `nbstripout` können die Metadaten und der Output von Jupyter Notebooks vor Commits entfernt werden, um das Repository sauber zu halten. Das Setup erfolgt über
```bash
nbstripout --install
```
